Por que aprender HTML5?

Com HTML5, você pode criar sites modernos, acessíveis e interativos para divulgar sua marca ou para fins lucrativos.

Spoiler: Em breve no curso você aprendera a usar comandos(tags) para melhores layouts! como por exemplo: <article>, <aside> e <section>

Curiosidades!

Além do HTML 5 você aprendera a utilizar tecnologias como: CSS, JavaScript e muito mais!.

O que é o HTML5?

HTML5 é a versão mais recente da linguagem de marcação HTML. Com ela, podemos criar páginas web mais ricas e interativas.

Exemplo básico da estrutura:

<!DOCTYPE html>
        <html>
        <head>
            <title>Exemplo</title>
        </head>
        <body>
            <h1>Olá, mundo!</h1>
        </body>
        </html>

Trabalhando com Listas

HTML oferece dois tipos principais de listas: não ordenadas e ordenadas.

Lista Não Ordenada (<ul>)

Usada quando a ordem dos itens não importa.

  • HTML
  • CSS
  • JavaScript

Lista Ordenada (<ol>)

Usada quando a ordem é importante, como em passos de um tutorial.

  1. Instalar um editor de código
  2. Aprender HTML básico
  3. Estudar CSS e JavaScript

Você também pode criar listas dentro de outras listas:

  • Frontend
    • HTML
    • CSS
  • Backend

Dicas importantes

  • Use sempre a declaração <!DOCTYPE html> no início do documento.
  • Utilize elementos semânticos para melhorar a acessibilidade aos leitores de tela.
  • Organize seu código para facilitar a leitura e manutenção.
  • Combine HTML com CSS e JavaScript para páginas mais completas.
  • Use <mark> para destacar trechos importantes.
  • Utilize <time> para representar datas e horários.
  • Experimente <kbd> para mostrar comandos de teclado no VS CODE: Ctrl + C
  • Utilize de extenções para tornar seu serviço mais rápido com o VS CODE
